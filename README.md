# Digital Rain Releases

Public installer assets for SSX360 Digital Rain — the local-first signing runtime for Matrix Scroll commit provenance.

This repository is the **public download surface** used by [ssx360.com](https://ssx360.com/). Installers are account-gated on the portal; release assets and `SHA256SUMS.txt` must stay reachable here for checksum verification.

**Install path:** Provisioned pilot and team accounts sign in at [ssx360.com/signup](https://ssx360.com/signup) → activate your device → download the installer from [ssx360.com/download](https://ssx360.com/download) → bind a workspace → every agent commit is signed locally before merge. Verify envelopes offline at [ssx360.com/verify](https://ssx360.com/verify).

**Team rollout:** [Book a provenance pilot](mailto:mission@ssx360.com?subject=Provenance%20Pilot%20Inquiry) for buyer-ready proof on protected branches. Installers are account-gated and must match SHA256SUMS.txt from the same GitHub Release — see [ssx360.com/trust](https://ssx360.com/trust).
